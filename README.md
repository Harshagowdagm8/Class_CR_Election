Quick Start
For Voters:
Open the application in your browser

Go to the "Voter Portal" tab

Enter any unique Voter ID (no pre-registration required)

Click "Verify" and select your preferred candidate

Confirm your vote

For Admins:
Go to the "Admin Panel" tab

Login with credentials:

Username: admin

Password: admin123

Create a new election with start/end dates

Add candidates with names and optional images/details

Share the generated election link with voters

Monitor votes in real-time

Admin Features
Election Management
Create new elections with custom titles and dates

Set active voting periods

End elections manually

Generate shareable election links

Candidate Management
Add/remove candidates

Upload candidate images

Track individual candidate votes

View which voters voted for each candidate

System Controls
Reset Current Election: Clear votes and candidates while keeping election structure

Delete Specific Election: Remove individual elections and all associated data

Delete All Elections: Wipe all election data

Full System Reset: Complete data wipe (use with caution)

Data Storage
The system uses browser localStorage to persist:

Election configurations

Candidate information

Vote records

Voter IDs (to prevent duplicate voting)
